# ReactiveKitCoreData

A ReactiveKit port of [RxCoreData](https://github.com/RxSwiftCommunity/RxCoreData)

## Installation

### CocoaPods

To install RxCoreData via CocoaPods, simply add the following line to your Podfile:

```ruby
pod "RxCoreData", "~> 1.0.0"
```
