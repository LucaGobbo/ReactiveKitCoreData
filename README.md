# ReactiveKitCoreData

A ReactiveKit port of [RxCoreData](https://github.com/RxSwiftCommunity/RxCoreData)

## Installation

### CocoaPods

To install ReactiveKitCoreData via CocoaPods, simply add the following line to your Podfile:

```ruby
pod "ReactiveKitCoreData", "~> 0.1.0"
```
